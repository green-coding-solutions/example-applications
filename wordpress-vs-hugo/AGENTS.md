# Agent Guide

This directory compares a static Hugo site with a minimal WordPress site. The runnable scenarios live in the child directories.

## Child Examples

- `hugo-apache`: static site served via Apache and accessed with `curl`.
- `wordpress-minimal`: minimal WordPress site, including version-pinned scenario variants.

## Agent Notes

- Choose the child directory based on whether the task is about the static or CMS side of the comparison.
- Read the child `AGENTS.md` before changing any flow files.
