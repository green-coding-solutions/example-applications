# Agent Guide

This directory groups Software Carbon Intensity examples. The runnable scenarios are in child directories, not here.

## Child Examples

- `idle`: SCI output for an idle workload.
- `simple-api`: SCI-oriented metric per API request.
- `static-algorithm`: SCI output for a compute-heavy workload.

## Agent Notes

- Start with the child directory that matches the workload type you need.
- Each child directory is self-contained and has its own `AGENTS.md`.
